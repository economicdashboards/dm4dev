A framework for using data mining and machine learning to uncover patterns in macroeconomic series

Approach:¶

- Fit polynomial regressions models to each individual series to find its trend¶

- Remove the trend from each series to make it stationary¶

- Apply data mining and machine learning algorithms on the de-trended series to uncover interesting patterns:¶

- Associative rules mining: when variables x and y are above their trend, z is also above its trend. Is there an interesting relation between x, y and z that we should explore?¶

- Classification and clustering: can we cluster countries in interesting groups using the de-trended series? What lessons for policy analysis?¶

- Machine learning: can we train a model to diagnose the situation of the country based on the data with an accuracy similar to the one of an expert?¶
